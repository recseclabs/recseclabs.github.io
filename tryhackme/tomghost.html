Offensive Pentesting: tomghost

Category: Write-up

Difficulty: Easy

Attack Box: Kali Linux

Exploit Jenkins to gain an initial shell, then escalate privileges by exploiting Windows authentication tokens.

Exploit a common misconfiguration on a widely used automation server, Jenkins. This tool is used to create continuous integration/continuous development pipelines that allow developers to automatically deploy their code once they made changes to it. After which, an interesting privilege escalation method will be used to get full system access.
